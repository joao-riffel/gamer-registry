# Perguntas

Responda de forma objetiva:

---

### ❓1 O que um teste automatizado verifica em uma API?
R: Verifica se a interface de programação de aplicações funciona conforme o esperado, analisando funcionalidades, confiabilidade, desempenho e segurança sem necessidade de interface de usuário.

---

### ❓2 Qual a diferença entre os erros:

- 400: R: O status 400 é um erro do cliente; O servidor não entende a requisição do front-end/usuário, pode ser por um erro de sintaxe ou dados inválidos.
- 500: R: O status 500 é um erro interno, de infraestrutura, como bugs no código falta de recursos ou permissões.

---

### ❓3 Por que testes automatizados são importantes no desenvolvimento de software?
R: Porque garantem a qualidade, velocidade e confiabilidade do produto, permitindo que as equipes entreguem funcionalidades com mais frequência e segurança.
